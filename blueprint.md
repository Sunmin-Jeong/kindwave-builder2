# Project Blueprint

## Overview

A simple web application to generate lottery numbers.

## Current Plan

### Features to be implemented:

*   **Lottery Number Generation:** Generate 6 unique random numbers between 1 and 45.
*   **Number Display:** Display the generated numbers in a visually appealing way.
*   **Generate Button:** A button to trigger the number generation.
*   **History:** Display a history of previously generated numbers.

### File Modifications:

*   **`index.html`:**
    *   Update the title to "Lotto Number Generator".
    *   Add the necessary HTML structure for the application.
*   **`style.css`:**
    *   Add styles for the application's components.
*   **`main.js`:**
    *   Implement the lottery number generation logic.
    *   Handle button clicks.
    *   Update the DOM with the generated numbers and history.
